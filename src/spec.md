# Specification

## Summary
**Goal:** Enable users to set and track daily calorie burn goals.

**Planned changes:**
- Add backend functions to store and retrieve daily calorie burn goals per user
- Create a frontend form for setting daily calorie goals with validation
- Display current goal and progress indicator in the dashboard

**User-visible outcome:** Users can set a daily calorie burn goal, view their current goal in the dashboard, and see their progress toward meeting it.
